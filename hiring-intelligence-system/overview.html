System Modules

5개의 모듈이 하나의 판단 구조를 완성

1. Role-Specific Interview Builder

JD · Track · Level 입력
  • 역량 매트릭스 자동 설계 및 가중치 조정
  • 코어 질문 + 꼬리 질문 + 깊이 단계 구성
  • 1~5점 루브릭과 좋은 답변 구조 정의
  • Red Flag 패턴 자동 생성
결과: 면접관 편차 최소화 · 공정성 확보

2. Resume-Based Interview Generator

후보자 주장 검증
  • Claim, Scope, Ownership, Impact 분해
  • 주장 검증 질문 자동 생성
  • Depth Escalation Engine (설명 → 구조 → 리스크)
  • Execution/Ownership Inflation Risk 탐지
결과: 이력서 주장을 검증하는 면접

3. 24h Feedback Reflection Model

과제 · 리뷰 · 수정 diff 분석
  • 1차 과제 → AI 리뷰 → 24h 수정
  • 반영률 · 개선의 질 · 리스크 추가 여부 측정
  • 커뮤니케이션 태도와 피드백 수용력 측정
결과: 성장 속도와 학습 가능성 측정

4. Risk Prediction Model

30~60일 리스크 예측
  • Early Exit, Communication, Execution, Feedback 리스크
  • 정량화된 리스크 스코어와 원인 신호
  • 불합격이 아니라 채용 전략 제안
결과: 리스크 기반 의사결정 프레임

5. End-to-End System Flow

JD 입력부터 리포트까지
  1. JD 입력 → Role Builder
  2. Resume 분석 → 질문 생성
  3. 면접 진행 → 구조화 노트
  4. Risk Engine → Candidate Report
결과: Hiring Intelligence OS 완성
