# Q2 2023 GranTON Application and Execution.

It's been six months since the launch of the Tokamak Network Grant Program GranTON, and in that time, we've seen a lot of great participation, which has spurred the development of the Tokamak Network Ecosystem.

11 project teams applied for 9 Project Grants and 2 Small Grants, totaling over $185,000 in grants. We're very grateful for everyone's interest in the Tokamak Network Grant Program, and we'd like to take this opportunity to provide a brief overview of the projects approved in Q2 2023.
